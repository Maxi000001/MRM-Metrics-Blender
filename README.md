# 📊 MRM Metrics & MRM Metrics Pro

![Blender](https://img.shields.io/badge/Blender-4.0%2B-orange?style=for-the-badge&logo=blender)
![Python](https://img.shields.io/badge/Python-3.10%2B-blue?style=for-the-badge&logo=python)

**MRM Metrics** — это мощный аналитический инструмент для Blender, предназначенный для точного измерения и контроля метрик сцены. Аддон помогает художникам и техническим специалистам следить за параметрами объектов в реальном времени.

---

## 🔥 Две версии — больше возможностей

### 🟦 MRM Metrics (Стандартная версия)
Базовый набор инструментов для повседневных задач:
* **Quick Measure:** Быстрое измерение расстояний и углов.
* **Scene Stats:** Отображение ключевых данных о мешах и полигонах.
* **Lightweight:** Минимальная нагрузка на систему.

### 🟧 MRM Metrics Pro (Профессиональная версия)
Расширенный функционал для сложного продакшена:
* **Advanced Analytics:** Глубокий анализ топологии и геометрии.
* **Custom Presets:** Сохранение собственных настроек и лимитов для метрик.
* **Real-time Feedback:** Мгновенное визуальное отображение критических параметров в вьюпорте.
* **Export Data:** Возможность выгрузки отчетов по метрикам сцены.

---

## 🛠 Установка
1. Скачайте нужную версию файла (`.py` или `.zip`) из этого репозитория.
2. В Blender: **Edit > Preferences > Add-ons > Install...**
3. Выберите файл и нажмите **Enable**.
4. Панель появится в N-панели (клавиша **N**) во вкладке **MRM Metrics**.
